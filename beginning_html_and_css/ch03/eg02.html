●「白蟒鞭法」： 使用极长的白蟒鞭，如灵蛇出洞，伸缩自如，灵动之极。

●「手挥五弦」： 伸五指在对方手肘轻轻一拂，使对方手臂微酸，全身消劲。

●「大伏魔拳」： 为《九阴真经》拳法中一招的名字，这拳法刚阳之气更重，与道家武学的一味阴柔并不相同，稳实刚猛之气，招数神妙无方，拳力笼罩之下，委实威不可当。（《神雕侠侣》中，周伯通以大伏魔拳与杨过的「黯然销魂掌」对拆）

●「摧坚神爪」： 威力奇大，先凝神运气，以十指撕抓拉击；下卷经文中内容提到：「五指发劲，无坚不破，摧敌首脑，如穿腐土。」，「摧敌首脑」是攻敌要害之意，飘忽灵动，变幻无方，但举手抬足之间却是正而不邪，态拟神仙。
